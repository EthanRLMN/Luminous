# Luminous Engine

A Game Engine Project made @ ISART Digital Paris in second year of Game Programming.
<br>


## Requirements
- **Windows-only**
  

- [CMake](https://cmake.org/download/) >= 3.20 & < 4.0.0
- [Git Bash](https://git-scm.com/downloads) >= 2.4.7
- [Visual Studio](https://visualstudio.microsoft.com/fr/vs/) 2022+ or [CLion](https://www.jetbrains.com/fr-fr/clion/download/#section=windows) 2024.3.4+
- [Vulkan SDK](https://vulkan.lunarg.com/sdk/home#windows) 1.4.304.X
<br><br>


## Libraries and APIs Used

- [Vulkan](https://docs.vulkan.org/spec/latest/index.html) v1.4.304
- [Assimp](https://github.com/assimp/assimp) v5.4.3
- [GLFW](https://www.glfw.org/docs/3.3/build_guide.html) v3.4
- [ImGUI](https://github.com/ocornut/imgui) v1.91.8
- [Jolt Physics](https://github.com/jrouwe/JoltPhysics) v5.2.0
- [stb_image](https://github.com/nothings/stb) v2.30
